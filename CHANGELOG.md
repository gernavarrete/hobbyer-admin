# Changelog — Hobbyer Admin Dashboard

### Agregado
- Overview con KPIs: usuarios, waitlist, grupos, matches
- Gestión de usuarios con búsqueda y paginación
- Detalle de usuario con hobbies y estadísticas
- Waitlist con invitación vía Firebase App Distribution (individual y masiva)
- Moderación: reportes, acciones, ban/suspend/activate
- Audit log de acciones de admins
- Gestión de grupos con cambio de status
- Stats de discovery (swipes y matches)
- Partners: listado y detalle
- Analytics: crecimiento, hobbies populares, funnel de conversión
- Feature flags para control de features en producción
- System health monitor
- Proxy Vercel para evitar Mixed Content con backend HTTP
- Auth con Cognito directo (USER_PASSWORD_AUTH flow)
